# Notify Bot

Бот который дудосит тебя уведами куда-то зайти и что-то сделать

## Установка

```bash
npm install
cp .env.example .env
```

## Запуск

```bash
# Development
npm run dev

# Production
npm start

# PM2
npm run pm2:start
```

## Конфигурация

| Переменная | Описание |
|------------|----------|
| `BOT_TOKEN` | Токен Telegram бота |
| `DEFAULT_URL` | URL по умолчанию |
| `REPEAT_INTERVAL_MS` | Интервал повторных уведомлений (мс) |
| `TIMEZONE` | Часовой пояс |

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/time` | Установить время напоминания |
| `/url` | Изменить ссылку |
| `/on` | Включить уведомления |
| `/off` | Выключить уведомления |
| `/status` | Показать настройки |
| `/stop` | Остановить повторные уведомления |
